# Javascript Functions

```javascript 
function add(a, b) { 
    return a + b
} 
const add = function(a, b) { return a + b } // Anonymous Function
const add = (a, b) => { return a + b }  // ES6 (new) Syntax
```

Three different ways of making a function!

# JavaScript Objects

```javascript
let a = {hello: 5, world: "hello there"}
a['hello'] // == 5
a.world // == "hello there"
a.instructors = ["Ethan", "Aivant"]
```

A set of key-value pairs where the keys are strings and values can be _anything_

# Functions as Object Values

```javascript
let calculator = {
    double: (x) => {return 2 * x}, 
    add: (x, y) => {return x + y}}
calculator.double(5) // == 10
calculator.subtract = function(x, y) { return x - y }
```

The values of an object can also be functions!

# Array.map

```javascript 
let numbers = [1, 2, 3, 4, 5]
let doubled = numbers.map((x) => {return 2 * x}) 
console.log(doubled) // [2, 4, 6, 8, 10]
console.log(numbers) // [1, 2, 3, 4, 5]
```

Array.map transforms an array of one thing into an array of another

---

# Array.filter

```javascript 
let numbers = [1, 2, 3, 4, 5]
even = numbers.filter((x) => {return x%2 == 0})
console.log(even) // [2, 4]
console.log(numbers) // [1, 2, 3, 4, 5]
```

Array.filter removes elements from an array that don't fit a condition
